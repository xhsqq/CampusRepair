# 校园报修系统 - 用户使用手册 (User Manual)

## 1. 系统概述
本系统旨在为在校学生和维修人员提供一个高效的报修与协作平台。

## 2. 角色功能指南

### 👤 学生用户 (Student)
- **注册与登录**: 使用学号注册，登录后进入“学生工作台”。
- **发起报修**: 点击“我要报修”，填写准确的故障地点（如：西区11舍402）和故障详情。
- **查看进度**: 在“报修列表”中实时跟踪工单状态（待接单、处理中、已完成）。
- **撤回报修**: 对于尚未被接单的错误报修，可以在详情页执行“撤回”操作。

### 🛠️ 维修人员 (Worker)
- **查看任务**: 进入“维修工工作台”，查看全校范围内的“待接单”任务。
- **执行接单**: 选择任务进入详情页，点击“我要接单”。
- **反馈结果**: 维修完成后，在详情页提交维修记录并将状态更新为“已完成”。

## 3. 常见问题 (FAQ)
- **为什么我无法撤回报修？**
  - 只有处于“待接单”状态的工单可以撤回。如果维修工已经接单，请联系相关部门。
- **如何修改个人信息？**
  - 点击右上角的“个人中心”即可修改姓名和联系电话。

## 4. 技术支持
如有系统崩溃或数据异常，请联系系统管理员。
